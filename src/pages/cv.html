Résumé Professionnel

Je suis un développeur web qui construit des sites fonctionnels et minimalistes. J'apprécie le code simple, bien structuré, et les outils qui font leur travail sans s'imposer. Je travaille principalement avec HTML, CSS, JavaScript et React.
